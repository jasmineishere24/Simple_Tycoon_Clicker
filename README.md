# Simple Tycoon Clicker

A small, single-file web tycoon / clicker game, ready to host on GitHub Pages.

## Features
- Click to earn coins.
- Upgrades: passive income, click improvements, multipliers, one-off upgrades.
- Leveling & simple XP.
- Prestige system (resets for permanent bonuses).
- Save/load via `localStorage`, export/import save file.
- Responsive and mobile-friendly.

## Files
- `index.html` — main UI.
- `style.css` — styles.
- `game.js` — game logic.
